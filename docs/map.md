#### Map    [demo点这里](https://github.com/baoendemao/javascript-summary/tree/master/demos/demo-map/map-1.js)
* 字典, 存放键值对
#### 常用函数
* new Map() 初始化
* set()   添加或者修改键值对
* size    长度
* delete()  删除指定的key
* has()   是否存在某个键
* get()   获取某个键的值
* forEach(function(value, key){})  获取键值对  
* keys()   获取键的相应的Iterator对象
